## 项目概述

### 项目名称
BOSS 直聘平台运营管理端（boss-manage-ui）

### 项目定位
面向平台运营管理人员的后台管理系统，负责企业、职位、求职者的审核与监管，内容运营和系统配置管理。

### 核心价值
- **平台治理**: 确保企业和职位信息真实可靠
- **风险管控**: 识别和处理违规行为
- **数据驱动**: 全面掌握平台运营状况
- **高效运营**: 提供完善的运营管理工具

### 目标用户
- 平台运营管理员
- 内容审核人员
- 客服管理人员
- 系统管理员

### 二、产品概述

#### 2.1 产品背景
随着平台规模的快速增长，需要建立完善的运营管理体系:
- 企业真实性审核需求
- 职位合规性监管需求
- 用户行为风控需求
- 平台数据可视化需求

#### 2.2 产品目标
1. **提升审核效率**: 自动化审核流程，减少人工成本
2. **降低平台风险**: 及时识别和处理违规行为
3. **数据驱动决策**: 全面掌握平台运营状况
4. **提高用户体验**: 营造公平、安全的招聘环境

#### 2.3 用户画像

##### 用户画像 1: 运营管理员 - 小王
- **年龄**: 28 岁
- **职业**: 平台运营管理员
- **痛点**: 
  - 每天需要审核大量企业认证
  - 难以快速识别虚假信息
  - 缺乏统一的管理工具
- **需求**:
  - 自动化的信息核验
  - 清晰的审核指引
  - 高效的批量操作

##### 用户画像 2: 风控专员 - 小李
- **年龄**: 26 岁
- **职业**: 内容审核专员
- **痛点**:
  - 海量会话难以逐一检查
  - 风险识别标准不统一
  - 处理流程繁琐
- **需求**:

  - 标准化的处理流程
  - 数据统计分析

### 四、详细功能说明

#### F202 - 企业认证审核

**功能描述**: 
审核企业提交的认证申请，包括资质材料审查和工商信息核验。

**优先级**: P0 - 必须实现

**用户故事**:
> 作为运营管理员，我希望高效审核企业认证申请，以便确保平台企业信息的真实性。

**功能详述**:

##### 输入要求
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 企业资质图片 | File[] | 是 | 营业执照、身份证等 |
| 企业信息 | Object | 是 | 工商登记信息 |
| 审核结果 | Enum | 是 | 通过/拒绝 |
| 拒绝原因 | String | 条件必填 | 拒绝时必填 |

##### 处理逻辑
```
1. 打开审核页面
2. 查看企业资质材料（支持放大）
3. 系统自动核验工商信息
4. 显示核验结果和风险点
5. 管理员做出审核决策
6. 如拒绝，选择拒绝原因
7. 提交审核结果
8. 更新企业状态
```

##### 输出结果
**通过场景**:
- 提示"审核通过"
- 企业状态变更为"正常"
- 发送通过通知

**拒绝场景**:
- 提示"已拒绝申请"
- 企业状态保持"待审核"
- 发送拒绝通知（含原因）

##### 验收标准
- [ ] 资质材料图片清晰展示
- [ ] 工商信息自动核验准确
- [ ] 风险点明确提示
- [ ] 审核操作流程顺畅
- [ ] 拒绝原因快速选择
- [ ] 审核结果正确保存

#### F501 - 会话审核

**功能描述**: 
对高风险的在线沟通会话进行质检和处理。

**优先级**: P1 - 应该实现

**用户故事**:
> 作为风控专员，我希望快速识别和处理高风险会话，以便维护平台沟通环境的安全。

**功能详述**:

##### 风险识别来源
1. **敏感词检测**: 自动识别敏感词汇
2. **用户举报**: 收到用户举报
3. **模型识别**: AI 风险识别模型
4. **异常行为**: 频繁发送相似消息

##### 风险等级定义
| 等级 | 颜色 | 说明 | 处理方式 |
|------|------|------|----------|
| 高 | 🔴 红色 | 严重违规 | 立即处理 |
| 中 | 🟠 橙色 | 可疑行为 | 优先审核 |
| 低 | 🟡 黄色 | 轻微异常 | 抽样检查 |

##### 审核流程
```
系统识别风险
    ↓
加入审核队列
    ↓
审核员查看会话
    ↓
判断是否违规
    ↓
【无违规】标记无风险 → 结束
    ↓
【有违规】选择处罚措施
    ↓
执行处罚 → 结束
```

##### 处罚措施
| 措施 | 适用场景 |
|------|----------|
| 警告 | 轻微违规 |
| 限制聊天 | 中度违规 |
| 临时封禁 | 严重违规 |
| 永久封禁 | 特别严重违规 |

##### 验收标准
- [ ] 风险等级准确标识
- [ ] 会话记录完整展示
- [ ] 敏感词高亮显示
- [ ] 处罚措施可选
- [ ] 审核结果可追溯
